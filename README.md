# DEEPACK Business Center — POS System

Mfumo kamili wa kusimamia biashara ya DEEPACK Business Center, Kigoma Tanzania.

## Vipengele
- Dashboard ya mauzo na faida
- Usimamizi wa bidhaa na stock
- POS — Uza na kutoa risiti
- Database ya wateja
- Ripoti na mwelekeo wa biashara
- Muunganiko na Google Sheets (Apps Script)
